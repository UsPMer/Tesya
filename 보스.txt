1 Stage - 우주

보스명 - 더스트(Dust)
공략법 - 특수블럭으로 더스트를 누르면 데미지
컨셉 - 우주 먼지들의 집합체 ..?(중2돋음)
	개처럼생김 검은색에 하얀눈동자 불타오르는듯한 느낌
스킬
0) 기본
- 기본적으로 캐릭터의 1/4배정도로 왕복함
- 기본적으로 블럭을 다뚫고 돌아다니며 블럭의 높이에만 영향
1) 축소
- 설명 : 캐릭터가 움직이는 영역이 줄어들고 더스트 또한 그 
	영역에서만 돌아다닌다. 영역이 줄시에 사라진 영역은
	반투명 밝은 회색을 씌운다.
- 모션 : 눈동자가 붉은색으로 바뀌고 소리지르는 모션이 나옴

2) 돌진
- 설명 : 더스트가 준비자세를 취한 후 앞으로 빠르게 돌진
	돌진하는 블럭은 5칸으로 일정
- 모션 : 돌진을 준비하는 자세를 취함

3) 블럭생성
- 설명 : 하늘에서 테트리스블럭이 랜덤으로 여러개가 떨어짐
- 모션 : 발을 구르는듯한 모션



2 Stage - 공장

보스명 - 관리자(Admin)
공략법 - 폭발로봇을 위쪽으로 유도해 보스근처에서 터트리면 데미지
컨셉 - 공장 전체를 제어하는 컴퓨터, 로봇같은 느낌
	맵의 일정높이에 TV처럼 걸려있음
스킬
0) 기본
- 기본적으로 캐릭터의 1/4배정도로 왕복하는 조그마한 로봇을 생성
- 로봇은 최대 3개이며 일정시간이 지나면 터지고 터지면서 블럭도 파괴
1) 레이저
- 설명 : 다방향에서 레이저가 나옴, 스테이지에 적용되는 레이저보다
	많은양, 블럭을 부숨
- 모션 : TV에 로딩창이 뜨며 로딩이 끝나면 레이저발사
2) 스파크
- 설명 : 폭발로봇들이 잠시 멈춰서 스파크를냄 스파크에닿으면 캐릭터가
	스턴상태에 빠짐
- 모션 : 화면에 전기표시가 뜸
3) 잔해낙하
- 설명 : 로봇이 폭발하면서 잔해를 수집하여 
	로봇의 잔해가 하늘에서 천천히 떨어짐
- 모션 : 톱니바퀴가 도는모션이 모니터에 나옴



3 Stage - 거울

보스명 - 도플갱어(Doppelganger)
공략법 - 파란색 가면일때 부딧히면 데미지
컨셉 - 자신의 캐릭터와 똑같이생긴 캐릭터가 보스이며 가면을쓰고있음

스킬
0) 기본
- 기본적으로 캐릭터의 1/2배정도로 왕복함
- 일정주기마다 가면의 색이 빨강 파랑으로 바뀜
- 기본적으로 블럭을 다뚫고 돌아다니며 블럭의 높이에만 영향
1) 복제
- 설명 : 일정시간동안 자신의 복제를 만들어냄
- 모션 : 갑자기 멈춰서서 앞으로 손을뻗음
2) 파편
- 설명 : 거울의 파편을 플레이어를 향해서 던짐
	3개이상 박히게되면 사망
- 모션 : 거울을 깨트리며 공중에서 파편을 던짐 
3) 반전
- 설명 : 파란색가면과 빨간색가면의 기능이 바뀜
- 모션 : 가면을 돌리는 모션
4) 거울블럭
- 설명 : 하늘에서 거울블럭이 하나 떨어져서 
	거울블럭에 닿으면 사망
- 모션 : 하늘로 손을 뻗는 모션



4 Stage - 겨울밤

보스명 - 럭스(Lux)
공략법 - 럭스의 행동력을 채워주는 배터리를 전부 파괴하면 클리어
컨셉 - 밝은 랜턴이 공중에 떠다니는 느낌
	특징은 공중유닛

스킬
0) 기본
- 기본적으로 캐릭터와 같은속도로 공중일정 높이에서 왕복함
1) 플레쉬
- 설명 : 잠깐동안 랜턴이 매우밝은빛을 쏴서 눈이 먼다는 느낌
	아무것도 보이지 않음
- 모션 : 잠시 멈춰서 랜턴이 한번에 밝아짐
2) 메테오
- 설명 : 하늘에서 불덩이가 떨어짐 떨어진위치는 폭발함
- 모션 : 랜턴에서 빛이 밖으로 도망가는듯한 모션
3) 과열
- 설명 : 랜턴이 과열되어 바닥으로 떨어짐
- 모션 : 갑자기 랜턴이 붉게 밝아짐
4) 레이저
- 설명 : 다방향에서 레이저가 나옴
- 모션 : 랜턴에서 빛이 밖으로 도망가는듯한 모션
